Incremental backups with rsnapshot.

Date Mon 10 February 2014 Tags incremental / backup / rsnapshot / rsync / cron

What is rsnapshot?

rsnapshot is a simple unix command-line tool for creating incremental backups. It is one of the tools you do not hear about much because it just does its job and lets you move on. I keep relying on year after year because it just works so well.

  • Available on almost every unix system and is likely already installed.
  • So small and lightweight that it works even on small NAS devices.
  • Takes only a few minutes to setup and just works.

How does it work?

As with all good unix command-line tools it does one job and does it well. Just configure which paths to backup (they can be from other servers via ssh as well), how many hourly/daily/weekly/monthly/yearly backups to keep.

As you would expect it will create differential incremental backups to save space and rotate incremental backups from hourly to daily etc.

You can then use cron to schedul when they are created and use rsync to push the backups to an offsite storage.
