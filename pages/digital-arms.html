Pelican

A static site generator, ideal for blogs and company sites.

The main advantages of static site generators

  • Simple cheap hosting almost anywhere (no database, server config, etc).
  • Quick development (Create content, generate and upload, done!).
  • Fast (only static files that can be heavily cashed).

Why Pelican?

  • It is popular, mature and has a rich feature set.
  • The documentation is very good.
  • Easy to setup, create content using Markdown and deploy.

I have not made an proper comparison with other static site generators and I am sure other system like Jekyll/Octopress are good as well.
